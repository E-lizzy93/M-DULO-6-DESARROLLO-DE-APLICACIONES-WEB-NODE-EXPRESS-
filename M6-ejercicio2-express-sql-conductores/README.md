# M6 - Ejercicio 2 Express + MySQL Conductores

Proyecto que conecta **Node.js con Express y MySQL** para mostrar una lista de conductores desde una base de datos.

## Tecnologías

* Node.js
* Express
* MySQL
* HTML / CSS / JavaScript

## Instalación

1. Clonar o descargar el repositorio.

2. Instalar dependencias:

npm install

3. Crear la base de datos en MySQL usando el archivo:

database.sql

4. Configurar la contraseña de MySQL en:

server.js

5. Ejecutar el servidor:

node server.js

6. Abrir en el navegador:

http://localhost:3000

## Estructura del proyecto

server.js → servidor Express
database.sql → base de datos
public/ → frontend (HTML, CSS, JS)

## Funcionalidad

* Mostrar conductores desde MySQL
* Conexión backend con Express
* Visualización en tabla HTML
